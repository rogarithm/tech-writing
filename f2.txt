Creating an order

In order to sell a product(s) to any customer, you must create an order in advance.

Keep the following steps to create an order:

Tap MENU () > Shopping Cart > Shopping Cart.  Then the registration screen appears.

Tap ▼ at the top and then select Salesperson. The login user is selected by default. If necessary, change the displayed salesperson.

Tap the search () icon for Customer and then select the desired customer.  It is not required. To register a new customer, tap &.

For more information about registration of a new customer, see Registering a new customer on page 16.

Tap the scan icon () to scan a barcode or search for a desired product.

Tap OK at the bottom.

Now the order is registered in the order list. The customer can wait until the requested product is ready.
}
