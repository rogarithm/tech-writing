절차서 구성 1

as> Make sure to charge the battery when initially using the product after purchase or when the product has not been used for a long period of time.

as> When sharing files via a mobile network, you may incur additional charges.

as> When clicking, the inquiry menu appears.

as> Access the Internet via a smartphone.

as> The password is factory set to '0000'.

as> Select Menu Language using the up/down direction button.
